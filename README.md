# Social-Spark-app

#Concise
:This app helps people that are not able or do not have the time, to have social interactions with others.

#Features:
*The app utilizes if statements(key logic) for recognition to processed data
*Feedback determined based on the input
*Use the of day to determine the response
*IF no input was add the app implemets "please provide a time" 
*Botton "Get Spark" is used to activate the reponse and generate a background (this feature was add using the setonclicklistener)
*Botton "Reset" is used to reset the textview to the original with a transparent background and no text output (setonclicklistener)

##Developemnt:

#CI/CD(confrim if the app runs)
